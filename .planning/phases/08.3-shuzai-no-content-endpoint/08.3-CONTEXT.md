# Phase 08.3 Context: 数载 Seedance 插件（禁用 content 下载接口）

## Goal

在不改宿主调用契约（`get_info/get_params/generate`）的前提下，基于 `video_plugin_zlhub_seedance` 交付新插件目录 `video_plugin_shuzai_seedance`，对接数载视频任务接口，并显式禁用 `GET /v1/videos/{task_id}/content` 下载路径。

## Hard Constraints

1. 仅允许以下接口：
   - `POST /v1/video/generations`
   - `GET /v1/video/generations/{task_id}`
2. 禁止实现/调用：
   - `GET /v1/videos/{task_id}/content`
3. 不接入图片审核链路（不保留 `AuditAESCipher`、审核参数、审核分支）。

## Inputs

- `docs/require3/shuzai-seedance2.0-video-api.md`
- `video_plugin_zlhub_seedance/main.py`
- `video_plugin_zlhub_seedance/ui/index.html`

## Outputs

- `video_plugin_shuzai_seedance/main.py`
- `video_plugin_shuzai_seedance/ui/index.html`
- Phase 08.3 planning artifacts

## Acceptance Snapshot

1. 新插件可被宿主识别并执行主链路。
2. 任务创建、轮询、下载闭环可运行。
3. 代码中不存在 `/v1/videos/{task_id}/content` 字符串与调用路径。
4. 审核相关参数与流程从 UI 和后端中移除。
