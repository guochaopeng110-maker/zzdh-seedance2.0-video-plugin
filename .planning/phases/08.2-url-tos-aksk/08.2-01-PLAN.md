# Phase 08.2 Plan 01: 固定参数与安全治理

## Objective

- **What:** 在 V2 插件内建立参数来源治理层，冻结核心 URL/TOS/AKSK 来源。
- **Why:** 当前存在 UI 可配与代码常量混合，容易引发配置漂移与安全暴露。
- **Output:** 可执行的来源策略、收敛后的 UI、治理回归测试与验证闭环。
