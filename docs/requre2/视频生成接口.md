# 视频生成接口V2

# 一、流程简介

视频生成任务接口是异步接口，视频生成任务流程：

1. 调用创建视频生成任务接口创建视频生成任务。
2. 创建任务时设置**callback_url**接任务的状态变化回调。
3. 定时使用查询接口查询视频生成任务状态（**可选，作为辅助，**建议任务创建10分钟后还未收到状态回调可以主动查询，每个任务每分钟查询一次，严禁频繁查询）
    1. 任务 running，过段时间再查询任务状态
    2. 任务完成，返回视频链接，在24小时内下载生成的视频文件

# 二、HTTP请求头

### Header 参数

| **参数名** | **类型** | **必填** | **说明** |
| --- | --- | --- | --- |
| Authorization | string | 是 | 鉴权 API_KEY，格式为 Bearer <API_KEY>，KEY请连续商务获取 |
| Content-Type | string | 是 | 固定为 application/json |
| X-Trace-ID | string | 否 | 请求跟踪ID，建议填写，用于出现问题时，反馈TraceID快速排查定位。每次请求必须不同随机32 位字符串，如 f3ab5c98d7e6f1a2b3c4d5e6f7a8b9c0 |

# 三、ZLHub接口地址

1、视频生成接口

`POST https://api.zlhub.cn/v1/task/create`

2、查询任务视频任务状态接口

`GET https://api.zlhub.cn/v1/task/get/{id}` 

3、取消或删除视频任务接口

`POST https://api.zlhub.cn/v1/task/cancel/{id}` 

接口差异说明：
1、为提高系统运行性能，ZLHub接口不支持base64格式的图片，请先将上传到云存储（TOS/OSS)等，获取公网链接后，再传到接口作为参考图片/素材。如果无云存储，可以找我们技术提供公共TOS云存储

### 6、视频任务响应体字段说明

### 该响应结构基于火山方舟 Seedance 2.0 原生返回扩展，**额外增加 zlhub平台`cost` 完整计费对象**，

### 顶层 `code`、`message` 为**我方平台新增字段**，火山方舟原始接口无此结构；

### `code` 用于标识接口整体业务状态，`message` 用于返回中文提示信息；

| **字段路径** | **字段名** | **类型** | **说明** |
| --- | --- | --- | --- |
| code | 响应码 | string | 接口状态，success 表示成功 |
| message | 提示信息 | string | 错误 / 提示文案，成功时为空 |
| data | 业务数据 | object | 视频生成结果主体 |
| data.content.last_frame_url | 末帧图片地址 | string | 视频最后一帧图链接，此处为 null |
| data.content.video_url | 视频地址 | string | 生成视频的火山引擎 TOS 下载链接（带签名，有时效） |
| **data.cost** | **计费信息** | **object** | **完整计费明细** |
| **data.cost.currency** | **货币单位** | **string** | **货币类型，CNY 为人民币** |
| **data.cost.input_cost** | **输入费用** | **string** | **输入内容计费金额** |
| **data.cost.output_cost** | **输出费用** | **string** | **生成视频计费金额** |
| **data.cost.total_cost** | **总费用** | **string** | **本次请求总花费** |
| data.created_at | 创建时间戳 | number | 任务创建时间（Unix 秒级时间戳） |
| data.draft | 是否草稿 | boolean | 是否为草稿任务，false 为正式任务 |
| data.duration | 视频时长 | number | 生成视频时长，单位秒 |
| data.execution_expires_after | 执行过期时间 | number | 任务执行超时时间，单位秒 |
| data.framespersecond | 帧率 | number | 视频帧率，FPS |
| data.generate_audio | 是否生成音频 | boolean | 是否附带生成音频 |
| data.id | 任务 ID | string | 本次视频生成唯一任务标识 |
| data.model | 模型名称 | string | 使用的生成模型 |
| data.ratio | 视频比例 | string | 画面宽高比 |
| data.resolution | 分辨率 | string | 视频分辨率 |
| data.seed | 随机种子 | number | 生成随机种子，用于复现结果 |
| data.service_tier | 服务等级 | string | 服务档位 |
| data.status | 任务状态 | string | 任务执行状态：queued 排队中 /running 运行中 /cancelled 已取消 /succeeded 成功 /failed 失败 /expired 超时 |
| data.error | 错误信息 | object / null | 错误提示对象，任务成功时返回 null，失败时返回错误详情 |
| data.error.code | 错误码 | string | 火山原生错误码（如 InvalidParameter、QuotaExceeded） |
| data.error.message | 错误提示信息 | string | 火山原生错误描述，会被平台翻译并封装到顶层 message 字段 |
| data.updated_at | 更新时间戳 | number | 任务最后更新时间戳 |
| data.usage.completion_tokens | 生成 tokens | number | 生成消耗 tokens |
| data.usage.total_tokens | 总 tokens | number | 本次总消耗 tokens |
