### 1. 视频生成任务,视频查询接口的主要变化:

**（1）修改视频生成接口地址**：将旧版地址替换为新版 `https://api.zlhub.cn/v1/task/create`。

**（2）修改任务查询状态查询接口**：将旧版查询地址替换为 `https://api.zlhub.cn/v1/task/get/{id} `。

**（3）（可选）添加追踪头**：在 HTTP 请求的 Header 中加入 `X-Trace-ID: <随机UUID>` ，后面接口调用问题可以提供traceid快速定位失败原因。

### 2. 素材审核接口的主要变化:

**鉴权方式重构**：取消了原有 AES-ECB 加密，统一改为标准的 Header Token 鉴权。

- **传参格式规范**：不再接受 Base64 格式的图片，必须提供公网可访问的图片 URL。
- **增加异步响应支持**：增加异步送审，callback回调或异步查询送审结果的机制。
- **服务地址变更**：域名迁移至 `https://asset.zlhub.cn`，响应 JSON 的层级进行了规范化调整。
